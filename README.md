# Asteroids from the Sky!

Using a tutorial to get more familiar with Pygame-CE. Code written while
following [Clear Code's tutorial on YouTube](https://www.youtube.com/watch?v=8OMghdHP-zs).

Code somewhat modified because I chose to use a `main()` function, which isn't
used in the tutorial.

Some resources not used because they aren't implemented in the game (yet).
